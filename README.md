# Stash NOC/SOC Availability Tiles

A small Stash override that adds availability tiles for AI services, media platforms, and a public Zabbix endpoint.

## Included tiles

### AI
- ChatGPT
- Claude
- Gemini
- Perplexity

### Media
- YouTube
- Netflix

### Monitoring
- Zabbix
